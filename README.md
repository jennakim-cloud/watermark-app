# 무신사 스탠다드 워터마크 삽입기

보도자료 이미지에 브랜드 로고를 자동으로 삽입하는 Streamlit 웹앱입니다.

## 기능

- 이미지 업로드 (JPG, PNG, WEBP 등 다중 파일 지원)
- 자동 규격 변환: **1056×720px**, JPG 출력
- 로고 삽입 위치 4가지 선택: 우상단 / 우하단 / 좌상단 / 좌하단
- 로고 색상 선택: 블랙 / 화이트
- 로고 크기 슬라이더 조절 (이미지 너비의 10%~40%)
- 개별 다운로드 / ZIP 일괄 다운로드

## 설치 및 실행

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 로고 파일 교체 방법

`logos/` 폴더 안의 파일을 실제 로고로 교체하세요.

| 파일명 | 용도 |
|--------|------|
| `무신사_스탠다드_우상단.png` | 우상단 위치용 로고 |
| `무신사_스탠다드_우하단.png` | 우하단 위치용 로고 |
| `무신사_스탠다드_좌상단.png` | 좌상단 위치용 로고 |
| `무신사_스탠다드_좌하단.png` | 좌하단 위치용 로고 |

> 블랙/화이트 버전을 별도로 관리하려면 `app.py`의 `LOGO_FILES` 딕셔너리에서
> `"black"`, `"white"` 키에 각각 다른 파일 경로를 지정하세요.

로고 파일은 **PNG (투명 배경, RGBA)** 형식을 권장합니다.
